Networking & Internet Terms

Subject Related Terms and Definitions
Network Classifications
  • LAN: Local area network, typically a connection of computers within a building or other individual physical location
  • MAN: Metropolitan area network, a connection of computers within a particular metropolitan area
  • WAN: Wide area network, network connecting computers over a large geographic area
Network models
  • Bus network: LAN setup where each node connects to a main cable called a bus
  • Star network: data network where all nodes are independently connected to one central unit
Protocols
  • CSMA/CD Ethernet: “Carrier-sense multiple access with collision detection” is an Ethernet control method used typically for local area networking. It uses carrier-sensing to defer transmissions until no other stations are transmitting
  • CSMA/CA Wifi: “Carrier-sense multiple access with collision avoidance” is a network access method where carrier sensing is used, but nodes attempt to avoid collisions by transmitting only after not detecting transmission
Combining Networks
  • Repeater: passes signals back and forth between the original two buses without considering the meaning of the signals when connecting networks. It regenerates the signal over the same network before the signal becomes too weak.
  • Bridge: passes signals back and forth, but unlike a repeater, only passes messages specifically meant for a certain computer. It filters content by reading the source and destination.
  • Switch: multiport bridge that can perform error checking before forwarding data
  • Router: device like a switch that routes data packets based on their IP addresses
  • Gateway: a passage to connect two networks that may work upon different networking models
Client/Server Model
  • P2P (Peer-to-Peer): file distribution where one person can receive a file from another and then provide that file to others
  • Distributed System: software units that execute as processes on different computers
  • Cloud Computing: a distributed system where independent computers work together to provide computation or services
Internet Addressing
  • Tier 1: ISPs (Internet Service Providers) with high-speed, high-capacity, international WANs. They are thought of as the backbone of the Internet.
  • Tier 2: comprise a lot of Internet infrastructure and act as backup, remaining connected even when Tier 1 fails. Tier 2 ISPs are typically regional or national providers.
  • Tier 3: access ISP, operated by a single authority that is in the business of supplying Internet access to individual homes and businesses. An example would be a company that provides Internet access to individuals within their organizations.
IP Address
  • ICANN:Internet Corporation for Assigned Names and Numbers, a nonprofit corporation established to coordinate the Internet’s operation by hierarchically assigning unique IP addresses
  • DNS: Domain name system, a collection of name servers that allows software to convert the address of a message into an IP address
  • TLD (Top-level Domain): Domain names describing the organization registering the domain, which reflects the domain’s classification. An example is edu to represent “educational.”
  • County Code TLDs:Domain names used to describe to county original of a site, such as .au for Australia
Internet Protocols
  • FTP: File Transfer Protocol, a standard communication protocol used for the transfer of computer files from a server to a client on a computer network
  • HTTP: The Hypertext Transfer Protocol, an application protocol for distributed, collaborative, hypermedia information systems that allows users to communicate data on Internet
  • SMTP: Simple Mail Transfer Protocol, a way that two computers on the network may interact when transmitting an email message from one host to the other
World Wide Web
  • URL: Uniform Resource Locator, information needed by a browser to contact the proper server and request the desired document
  • Client-side Activities: activities if they are performed by a client (such as a browser)
  • Server-side Activities: activities if they are performed by a server (such as a webserver)
Internet Four-layer Software
  • Application Layer: an abstraction layer that specifies the shared protocols and interface methods used by hosts in a communications network
  • Transport Layer: provides communication between application processes by collecting message segments from applications, and transmits them into the network
  • Network Layer: part of the Internet communications process where connections occur by sending packets of data back and forth between different networks
  • Link Layer: protocol layer that transfers data between nodes on a network segment across the physical layer
Internet is:
  • Packet-switching Network: transmits data across digital networks by breaking it down into blocks or packets for more efficient transfer using various network devices
  • Fault Tolerant: ability of a system (computer, network, cloud cluster, etc.) to continue operating without interruption when one or more of its components fail
  • Redundant: Network redundancy is a communications pathway that has additional links to connect all nodes in case one link goes down.
Cybersecurity
  • Malware: malicious software
  • Virus: software that infects a computer by inserting itself into programs that already reside in the machine
  • Worm: an autonomous program that transfers itself through a network, taking up residence in computers and forwarding copies of itself to other computers
  • Trojan Horse: a program that enters a computer system disguised as a desirable program, such as a game or useful utility package, that is willingly imported by the victim. Once downloaded, however, the Trojan horse performs additional activities that might have harmful effects.
  • Spyware: software that collects information about activities at the computer on which it resides and reports that information back to the instigator of the attack
  • Phishing: technique of obtaining information fraudulently by tricking the victim into thinking the message is from a trusted source
  • DDos Attack: Distributed denial of service attack, where after a signal is given, many computers (sometimes called a botnet) swamp the target with messages
Protection
  • Firewall: malware prevention technique that filters traffic passing through a point in the network
  • Spam Filter: firewalls designed to block unwanted email
  • Proxy Server: a software unit that acts as an intermediary between a client and a server with the goal of shielding the client from adverse actions of the server
  • Antivirus Software: software used to detect and remove the presence of known viruses and other infections
  • Cryptography: the study of sending and receiving messages securely in the presence of threats
  • Public Key Encryption: techniques within the field of cryptography where encryption and decryption algorithms are known, and one of the keys is known, but the encoded message remains secure
